# Changelog

本文档记录「投资观察站」项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，

## [0.2.0] — 2026-07-13

### 新增
- `S.apiUrl()` 函数统一管理所有 API 请求地址，支持静态模式和动态模式自动切换
- GitHub Pages 部署完整流程（`.github/workflows/deploy.yml`）
- 静态模式下自动降级使用本地 JSON 数据
- `build_data.py` 脚本用于 CI 构建时预生成静态数据文件

### 修复
- 修复静态模式下 API URL 硬编码导致的数据加载失败
- 修复 `_config.yml` 文件 BOM 头导致 GitHub Pages 构建异常
- 修复服务端状态文本显示问题

### 变更
- 所有 API 请求统一通过 `S.apiUrl()` 代理
- 数据文件变更自动触发 GitHub Actions 重新部署

## [0.1.0] — 2026-07-10

### 新增
- **四因子看板**：市场情绪、流动性、周期位置、龙头强度四大维度实时展示
- **产业逻辑页面**：产业链上下游关系可视化（供应链→制造→模组→终端）
- **复盘工具**：基于阈值规则的盘后诊断系统（`review_rules.py`）
- **量能监测**：实时成交量异常监控与历史对比
- **盘后复盘**：自动化复盘总结生成（含情绪周期/幸存者偏差/竞价分析）
- **主线赛道五维量化打分引擎** (`scoring.py`)：
  - 五维：供需缺口(25%) + 订单能见度(25%) + 业绩兑现(20%) + 政策红利(15%) + 中长期资金(15%)
  - 5条主线赛道：AI算力配套、存储芯片、光芯片/光模块、国产半导体设备、华为昇腾产业链
- **实时数据服务** (`dataservice.py`)：基于东方财富公开API的行情数据拉取
- **因子引擎** (`factors_engine.py`)：四因子实时计算
- **情绪预警子系统** (`sentiment-warning-system/`)：独立的市场情绪监控页面
- 深色主题 UI 设计（`index.html` + `app.js`）
- 中英文双语界面 (`i18n.json`)
- 离线/在线模式自动切换
- 数据缓存与快照机制
- 产业链配置 (`industry_chain_config.json`)
- 跟踪配置 (`tracking_config.json`)
