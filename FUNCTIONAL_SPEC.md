# 投资观察站 · 功能说明

> 本文件仅描述项目的功能与业务逻辑，不涉及界面与视觉效果。

---

## 一、项目定位

投资观察站是一套面向 A 股短线交易的**实时监控与决策辅助系统**，整合四大核心因子与主线赛道五维量化打分引擎，帮助交易者快速把握市场节奏。

- 数据源：东方财富公开 API（实时行情、涨跌停、北向资金等）
- 后端：Python 3（纯标准库，零依赖），HTTP 数据服务器（端口 8765）
- 前端：Vanilla JavaScript + ECharts
- 部署：GitHub Pages + GitHub Actions
- 缓存：内存缓存 + JSON 文件快照

---

## 二、核心模块功能

### 1. 四因子看板

实时计算并展示四大因子，综合推演市场概率分：

| 因子 | 权重 | 含义 |
|------|------|------|
| 情绪周期 | 35% | 市场情绪所处的阶段（冰点/退潮/主升/高潮等） |
| 板块联动 | 30% | 主线板块之间的协同强度 |
| 筹码结构 | 20% | 资金与筹码的分布健康度 |
| 隔夜预期 | 15% | 隔日消息与外围环境预期 |

**输出信息：**
- 综合概率分（0–100）与仓位建议（空仓/轻仓/半仓/重仓）
- 风险等级（高/中/低）
- 较昨日对比变化
- 核心操作建议与禁忌提示
- 涨停数、跌停数、炸板率、成交额、北向资金、最高连板等六项市场指标
- 递进约束传导：情绪→板块→筹码→隔夜，逐级约束逻辑
- 情绪趋势时序图

支持两种模式切换：**趋势波段** / **龙头打板**。

### 2. 产业逻辑

可视化产业链上下游关系，支持供应链→制造→模组→终端追踪：

- 左侧赛道列表（当前预置：AI 算力产业链）
- 右侧提供两种视图：
  - **链状视图（桑基图）**：展示上游供给端 → 中游制造运营 → 下游需求应用的资金/逻辑流向
  - **树状视图（树图）**：层级展开，可漫游
- 点击节点展开详情面板：产业逻辑、核心壁垒、核心标的、近期资讯
- 节点按景气度（0–100）着色，区分强/中/弱

### 3. 复盘工具

基于阈值规则的盘后个股诊断：

- 实时/收盘个股列表（代码、名称、最新价、涨跌幅、成交额、换手率、所属板块）
- 筛选条件：代码/名称关键字、最低成交额、最低换手率
- 一键导出 CSV 复盘报告

### 4. 量能监测

实时成交量异常监控：

- 监测当日放量个股，计算**放量倍数 = 当日成交量 / 前 5 日最小单日成交量**
- 放量超过 3 倍高亮标记
- 今日触发的股票置顶显示
- 数据状态区分：实时 / 收盘数据 / 昨日数据 / 暂无数据
- 筛选：代码/名称、最低放量倍数
- 默认每 5 分钟自动刷新

### 5. 盘后复盘（结构化生成）

基于固定模板的结构化复盘，纯本地规则计算，分三个子模块：

- **盘后快速复盘**：汇总大盘指数、涨跌家数、涨跌停、情绪阶段、核心题材、风向标
- **次日作战计划**：输入目标股与风控阈值，结合当前成交额与情绪给出计划
- **盘中突发诊断**：输入持仓与当前情况，按情绪强弱、板块支撑、盈亏幅度给出处置建议

支持一键复制生成内容。

### 6. 历史数据中心

支持按日期范围回溯历史数据，四个标签页：

- **因子趋势**：综合评分与四因子历史时序表 + 趋势折线图
- **量能异动**：历史放量个股记录
- **市场快照**：每日上证/深证/创业板涨跌、成交额、涨跌停、炸板率、北向资金
- **板块排行**：历史板块排名、涨跌、净流入、股票数

数据来源：Supabase（在线）或本地缓存（离线）。

---

## 三、五维打分引擎

主线赛道量化评分系统，覆盖五条主线、十五项指标：

- 评分维度：供需 / 订单 / 业绩 / 政策 / 资金
- 覆盖赛道示例：AI 算力配套、存储芯片、光芯片/光模块、国产半导体设备、华为昇腾产业链

---

## 四、通用功能

- **实时/离线双模式**：未连接服务器时自动降级为离线模式，使用本地缓存 JSON 数据
- **中英文切换**：界面文案双语（i18n.json）
- **主题切换**：深色 / 浅色
- **自动轮询**：看板每 10 秒刷新，量能每 5 分钟刷新
- **服务器健康检查**：每 30 秒探测在线状态与数据库连接

## 五、数据接口（后端 API）

| 端点 | 说明 |
|------|------|
| `/api/ping` | 服务存活检测 |
| `/api/factors` | 四因子实时数据 |
| `/api/factors/history` | 情绪趋势历史 |
| `/api/volume/monitor` | 量能监测 |
| `/api/review/summary` | 盘后复盘汇总 |
| `/api/review/stocks` | 复盘个股列表 |
| `/api/stock/search` | 股票搜索 |
| `/api/history/*` | 历史因子/量能/快照/板块 |
| `/api/supabase/status` | 数据库状态 |
