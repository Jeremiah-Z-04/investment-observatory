# 投资观察站 · 视觉传达信息总结

> 本文件仅描述新 UI 前端系统**通过视觉传达了什么信息**，不涉及技术实现细节。

---

## 1. 产品身份与定位

视觉系统整体传达出一个核心信息：

**「这是一套专业、冷静、可信赖的 A 股短线交易座舱，而非娱乐化的行情软件。」**

- 深色座舱 + 玻璃拟态 → 传达「仪器感」「专业终端」而非「社交 App」
- 留白与克制 → 传达「信息密度高但有序」，区别于泛 AI 深色霓虹风
- 汕头游资专用 → 视觉为高频、快节奏决策服务，强调即时可读

---

## 2. 色彩语义传达

### 2.1 品牌主色
- **蓝紫渐变 (#4F7DF3 → #7A9DF8)**：只用于品牌标识、主操作、高亮焦点
- 传达：科技、理性、决策中枢；克制使用（≈10% 面积）以保持权威感

### 2.2 市场语义色（中国惯例）
- **红 (#FF4D4F) = 涨**：贯穿所有涨跌、强度、风险信号
- **绿 (#22C55E) = 跌**：与红形成强对比，一眼可辨方向
- 传达：符合本土交易者直觉，零学习成本

### 2.3 状态色
- **黄 (#F59E0B) = 中性 / 警告**
- **青 (#38BDF8) = 信息提示**
- 传达：分级预警，颜色即优先级

### 2.4 表面色
- 半透明深色表面 + 顶部高光 → 传达「悬浮的玻璃面板」层次感
- 传达：信息分层，不靠重阴影而是在材质上做区分

---

## 3. 材质与空间传达

- **玻璃拟态（blur + 低透白边）** → 传达「悬浮、轻盈、现代」
- **圆角体系（8/12/16/24px）** → 传达「柔和、无攻击性、iOS 亲和力」
- **4pt 间距网格** → 传达「秩序、节奏、可预测」
- **卡片悬浮微抬升 + 边框提亮** → 传达「可交互、有反馈、活的信息」
- **顶栏 sticky + 模糊** → 传达「恒定锚点，导航永远在握」

---

## 4. 字体与层级传达

- **SF Pro / PingFang 系统字体栈** → 传达「原生、即时、无加载感」
- **数字 tabular-nums 等宽** → 传达「数据可对齐比较、可信」
- **固定字号比例（非流式）** → 传达「座舱 UI 的空间确定性」
- 层级通过「尺寸 + 字重 + 颜色 + 留白」四维叠加 → 传达「谁主谁次，一眼分明」

---

## 5. 动效语言传达

动效本身是一种信息语言：

- **Expo 缓出入场（500ms, staggered）** → 传达「内容有秩序地登场，而非闪现」
- **数值滚动 + 脉冲** → 传达「数据在实时呼吸，是活的信号」
- **弹簧按压反馈** → 传达「界面有物理质感，可触摸」
- **Tab 指示器滑动** → 传达「位置连续性，当前所处明确」
- **页面切换淡入上滑** → 传达「上下文延续，不割裂」
- **减少动效模式** → 传达「尊重敏感用户，可用性优先」

整体传达：**「流畅、自然、有重量感的非线性运动」—— 对标 Apple 的体感。**

## 6. 各模块视觉信息

| 模块 | 视觉传达的核心信息 |
|------|-------------------|
| **四因子看板** | 综合概率分作为视觉主角（大号数字 + 辉光），四因子卡片网格环绕 → 传达「一眼判断仓位与情绪周期」 |
| **量能监测** | 表格 + 放量倍数高亮（红） → 传达「异常即警报，启动信号优先」 |
| **产业逻辑** | 产业链桑基/树状图，节点按景气度着色 → 传达「上下游传导与主线强弱」 |
| **复盘工具** | 高密度表格 + 一键导出 → 传达「盘后快速筛选与归因」 |
| **盘后复盘** | 三段式 Tab（复盘/计划/诊断）+ 结构化输出 → 传达「模板化、可复制的决策流程」 |
| **历史数据** | 趋势图 + 快照表 → 传达「回溯验证，信号可证」 |

---

## 7. 响应式与无障碍传达

- **桌面侧边栏 / 移动端底部 Tab** → 传达「多端一致的操作锚点」
- **44px 触控区** → 传达「移动端可盲操」
- **焦点环 + 对比度达标** → 传达「人人可用，不排斥」
- **减少动效支持** → 传达「克制即专业，不炫技」

---

## 8. 差异化记忆点（一句话）

> **「红涨绿跌」作为中国市场的视觉母语，配合玻璃质感的 iOS 级非线性动效，让这套交易座舱在众多深色面板中，既专业可信，又令人记住。」**
