# 投资观察站 · 设计上下文

## 1. 目标用户

- **Who**: A 股短线交易者、游资风格个人投资者
- **情境**: 盘中快速扫盘、复盘回顾、板块/产业链追踪
- **核心诉求**: 在噪声中提取信号，快速判断情绪周期与资金流向

## 2. 使用场景

| 场景 | 频率 | 关键行为 |
|------|------|----------|
| 四因子看板 | 高频 | 扫一眼综合分与四大因子，决定仓位 |
| 量能监测 | 盘中 | 发现放量异动股，捕捉启动信号 |
| 产业逻辑 | 中频 | 理解产业链上下游，锁定主线细分 |
| 复盘工具 | 收盘后 | 生成结构化复盘文本 |
| 历史数据 | 低频 | 回测趋势、验证策略 |

## 3. 品牌调性

- **Tone**: 沉稳、精准、高级、专注
- **类比**: Apple Stocks + 专业交易终端的折中
- **关键词**: 冷静、克制、有呼吸感、数据可信、即时反馈

## 4. 视觉方向

- 深色座舱为主，辅以柔和浅灰表面
- 色彩点缀克制，只在涨跌与关键状态上使用语义色
- 大量使用留白、圆角、微弱玻璃质感
- 动效自然、非线性、以 Apple 弹性曲线为基准

## 5. 动效原则

- 入口：staggered 上浮 + 淡入（500ms，spring 弹性收尾）
- 状态切换：缓出指数曲线（cubic-bezier(0.16, 1, 0.3, 1)）
- 数值更新：数字滚动 + 色闪反馈
- 页面切换：共享元素过渡 + 淡入淡出
- 尊重 prefers-reduced-motion

## 6. 差异化记忆点

- 中国市场涨跌语义色（红涨绿跌）作为设计语言贯穿始终
- 所有卡片与控件都有 iOS 级别的按压反馈与悬浮抬升
- “呼吸式”数据刷新：非打断的轻柔脉冲，而不是刺眼闪烁
